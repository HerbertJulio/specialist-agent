Privacy Policy ​

Last updated: March 2, 2026

Overview ​

Specialist Agent is an open-source CLI tool and plugin for AI-powered code editors. This Privacy Policy explains how data is handled by the tool itself and by this documentation website.

The CLI Tool ​

Specialist Agent operates entirely on your local machine. The tool:

  • Does not collect, transmit, or store any personal data
  • Does not include telemetry or usage tracking
  • Does not communicate with any external servers on its own
  • Processes all files and code locally within your development environment

AI Provider Communication ​

When you use Specialist Agent through an AI-powered editor (Claude Code, Cursor, VS Code, etc.), your prompts and code context are sent to the AI provider (e.g., Anthropic, OpenAI) by the editor — not by Specialist Agent. These communications are governed by the respective AI provider's privacy policy:

  • Anthropic Privacy Policy
  • OpenAI Privacy Policy

Documentation Website ​

This documentation website (specialistagent.com.br) may use analytics services to understand how visitors interact with the content. The data collected is anonymous and may include:

  • Pages visited and navigation patterns
  • Approximate geographic location (country/region level)
  • Device type, browser, and operating system
  • Referral source
  • Time spent on pages

This data is used solely to improve the documentation. No personally identifiable information is collected through analytics.

Cookies ​

This website may use cookies for:

  • Analytics: To measure website traffic and usage patterns
  • Preferences: To remember your language selection (English / Portuguese)

You can disable cookies through your browser settings at any time.

Hosting ​

This website is hosted on GitHub Pages. GitHub may collect technical data such as IP addresses for security and operational purposes. See GitHub's Privacy Statement for details.

Third-Party Services ​

Specialist Agent is distributed via npm. When you install the package using npm install or npx, npm may collect usage data as described in their privacy policy.

Open Source ​

Specialist Agent is released under the MIT License. The full source code is publicly available for review on GitHub.

Children's Privacy ​

This tool and website are intended for software developers and are not directed at children under 13 years of age.

Changes to This Policy ​

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated "Last updated" date. For significant changes, a notice may be posted on the repository.

Contact ​

If you have questions about this Privacy Policy, please open an issue on GitHub or contact the maintainer through their GitHub profile.
