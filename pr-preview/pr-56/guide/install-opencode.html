Install ​

Or manually copy from the repo to .claude/.

Use ​

Reference agents in prompts:

Using @builder:
Create a UserProfile component with avatar and bio fields.

Using @tdd:
Implement calculateTax following TDD.

Using @planner:
Plan the checkout flow implementation.
