Política de Privacidade ​

Última atualização: 2 de março de 2026

Visão Geral ​

Specialist Agent é uma ferramenta CLI open-source e plugin para editores de código com IA. Esta Política de Privacidade explica como os dados são tratados pela ferramenta e por este site de documentação.

A Ferramenta CLI ​

O Specialist Agent opera inteiramente na sua máquina local. A ferramenta:

  • Não coleta, transmite ou armazena dados pessoais
  • Não inclui telemetria ou rastreamento de uso
  • Não se comunica com servidores externos por conta própria
  • Processa todos os arquivos e código localmente no seu ambiente de desenvolvimento

Comunicação com Provedores de IA ​

Quando você usa o Specialist Agent através de um editor com IA (Claude Code, Cursor, VS Code, etc.), seus prompts e contexto de código são enviados ao provedor de IA (ex: Anthropic, OpenAI) pelo editor — não pelo Specialist Agent. Essas comunicações são regidas pela política de privacidade do respectivo provedor:

  • Política de Privacidade da Anthropic
  • Política de Privacidade da OpenAI

Site de Documentação ​

Este site de documentação (specialistagent.com.br) pode utilizar serviços de analytics para entender como os visitantes interagem com o conteúdo. Os dados coletados são anônimos e podem incluir:

  • Páginas visitadas e padrões de navegação
  • Localização geográfica aproximada (nível de país/região)
  • Tipo de dispositivo, navegador e sistema operacional
  • Fonte de referência
  • Tempo gasto nas páginas

Esses dados são usados exclusivamente para melhorar a documentação. Nenhuma informação pessoal identificável é coletada através do analytics.

Cookies ​

Este site pode utilizar cookies para:

  • Analytics: Para medir o tráfego e padrões de uso do site
  • Preferências: Para lembrar sua seleção de idioma (Inglês / Português)

Você pode desativar cookies através das configurações do seu navegador a qualquer momento.

Hospedagem ​

Este site é hospedado no GitHub Pages. O GitHub pode coletar dados técnicos como endereços IP para fins de segurança e operação. Consulte a Declaração de Privacidade do GitHub para mais detalhes.

Serviços de Terceiros ​

O Specialist Agent é distribuído via npm. Quando você instala o pacote usando npm install ou npx, o npm pode coletar dados de uso conforme descrito em sua política de privacidade.

Código Aberto ​

O Specialist Agent é publicado sob a Licença MIT. O código-fonte completo está disponível publicamente para revisão no GitHub.

Privacidade de Menores ​

Esta ferramenta e site são destinados a desenvolvedores de software e não são direcionados a menores de 13 anos.

Alterações nesta Política ​

Podemos atualizar esta Política de Privacidade periodicamente. As alterações serão refletidas nesta página com a data de "Última atualização" atualizada. Para mudanças significativas, um aviso poderá ser publicado no repositório.

Contato ​

Se você tiver dúvidas sobre esta Política de Privacidade, abra uma issue no GitHub ou entre em contato com o mantenedor através do seu perfil no GitHub.
